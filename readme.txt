v1.1.0更新日志
修复了一些bug，增加了放逐伊甸
放逐之后：
如果已经解锁，则pc可以独享伊甸小屋
伊甸不会再来森林追pc了
理论上伊甸不会来镇上抓pc
伊甸不再可以从精神病院救出pc
贝利的惩罚链会跳过伊甸

v1.2.0更新日志
修复了一些文本效果
增加了放逐约旦（比我想的复杂得多）和仅取消贞洁检测
放逐之后：
部分技术及时间原因导致约旦放逐后pc不能忏悔自己的破誓，不好意思（哭）
神殿不再能找到约旦
周日的弥撒被取消了（没人主持）
在一些小剧情中把约旦替换掉了（比如不能看约旦洗澡，别人送pc去医院等）
约旦不会再来发布救人和圣矛任务，已接受不影响，但在约旦被放逐的时候这两个任务无法完成最后阶段
不再有贞洁检查，相应的也没有神殿补贴了
pc可以在神殿躲债，但是如果是第一次触发（就是贝利还没有要钱，只是告诫的那一次），贝利和约旦的交流仍然需要约旦未被放逐
悉尼的承诺仪式无法举行
悉尼的堕落仪式也无法举行（我实在想不出来合理的剧情，除了约旦还能有谁来检查？）
凯拉尔庄园约旦和贝利救人的事件摆烂了，除非我想出来了或者看到了合适的除了贝利之外可以救pc的人
出于功能性考虑加入了仅取消贞洁检测的功能
